Dokumentacja Triggerów

Opis triggerów wywołujących funkcję aktualizacji średniej oceny.

Trigger: update_podcast_avg_rating

Definicja Triggera


CREATE TRIGGER update_podcast_avg_rating
AFTER INSERT ON Podcasts_ratings
FOR EACH ROW
EXECUTE PROCEDURE update_avg_rating();

Opis

Ten trigger wywołuje funkcję update_avg_rating() po dodaniu nowej oceny dla podcastu, co pozwala na natychmiastową aktualizację średniej oceny w tabeli Podcasts.

Trigger: update_album_avg_rating

Definicja Triggera


CREATE TRIGGER update_album_avg_rating
AFTER INSERT ON Albums_ratings
FOR EACH ROW
EXECUTE PROCEDURE update_avg_rating();

Opis

Trigger wywoływany po wstawieniu nowej oceny dla albumu. Dzięki niemu średnia ocena albumu w tabeli Albums jest aktualizowana.

Trigger: update_song_avg_rating

Definicja Triggera


CREATE TRIGGER update_song_avg_rating
AFTER INSERT ON Songs_ratings
FOR EACH ROW
EXECUTE PROCEDURE update_avg_rating();

Opis

Ten trigger odpowiada za aktualizację średniej oceny dla utworu w tabeli Songs po dodaniu nowej oceny.
